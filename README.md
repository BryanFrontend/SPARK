# Assets

This directory contains visual assets for the Spark project.

## Contents

- `logo/spark_logo.png` — Spark lightning bolt logo (512x512 PNG)
- `diagrams/architecture.png` — System architecture diagram
- `screenshots/terminal_demo.png` — Terminal demo screenshot

## Logo Usage

The Spark logo may be used in:
- Forks of this repository (with attribution)
- Blog posts and articles about the project
- Social media posts referencing the project

Please do not use the logo to imply endorsement by OpenClaw Labs.
